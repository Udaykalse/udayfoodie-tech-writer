# Frequently Asked Questions (FAQ)

## Overview

This document answers common questions about the **UdayFoodie Landing Page** project.

---

## 1. What is UdayFoodie Landing Page?

UdayFoodie Landing Page is a static front-end website developed using HTML5 and CSS3. It showcases a modern and responsive landing page for a fictional food service business.

---

## 2. Does this project require a backend?

No.

This is a static website and does not require a backend server, database, or API.

---

## 3. Which technologies are used?

The project uses:

- HTML5
- CSS3

---

## 4. How do I run the project?

Clone the repository:

```bash
git clone https://github.com/Udaykalse/udayfoodie-landing-page.git
```

Then open `index.html` in your web browser.

Alternatively, use the **Live Server** extension in Visual Studio Code.

---

## 5. Can I customize the design?

Yes.

You can modify:

- Colors
- Fonts
- Images
- Layout
- Sections
- CSS styles

---

## 6. Where are the images stored?

All website images are located in:

```text
assets/Images/
```

---

## 7. Is the website responsive?

Yes.

The layout is designed to adapt to different screen sizes, including desktop, tablet, and mobile devices.

---

## 8. How can I contribute?

Fork the repository, create a new branch, make your changes, and submit a Pull Request.

Refer to `docs/contributing.md` for contribution guidelines.

---

## 9. Can I use this project for learning?

Yes.

This project is suitable for beginners learning:

- HTML
- CSS
- Responsive Design
- Git
- GitHub
- Technical Documentation

---

## 10. Where can I report issues?

If you discover bugs or have suggestions, create an Issue in the GitHub repository with:

- A clear description
- Steps to reproduce
- Expected behavior
- Screenshots (if applicable)

---

## Summary

This FAQ provides quick answers to common questions, helping developers and users understand the project without reading the full documentation.
