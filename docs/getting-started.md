# Getting Started

## Welcome

Welcome to the **UdayFoodie Landing Page** documentation.

This guide helps new developers understand the project structure and begin making changes with minimal setup.

---

## Open the Project

After cloning the repository, open the project folder in Visual Studio Code.

```bash
code .
```

---

## Project Files

The project contains only a few files:

- `index.html` — Main HTML page
- `style.css` — Styling for the website
- `assets/Images/` — Images used throughout the website
- `docs/` — Project documentation
- `images/` — Screenshots used in documentation

---

## Making Your First Change

As an example:

1. Open `index.html`.
2. Change the website title.
3. Save the file.
4. Refresh your browser.

You should immediately see the updated title.

---

## Editing Styles

Open:

```text
style.css
```

Modify properties such as:

- Colors
- Fonts
- Spacing
- Layout
- Buttons

Refresh the browser to preview changes.

---

## Working with Images

All website images are stored in:

```text
assets/Images/
```

To replace an image:

1. Add the new image.
2. Update the image path in `index.html`.
3. Refresh the page.

---

## Best Practices

- Keep HTML semantic.
- Use meaningful class names.
- Organize CSS logically.
- Optimize image sizes.
- Test responsiveness after changes.
