# Installation Guide

## Overview

This guide explains how to set up and run the **UdayFoodie Landing Page** on your local machine.

---

## Prerequisites

Before starting, ensure you have:

- A modern web browser (Chrome, Firefox, Edge, or Brave)
- Git (optional, for cloning the repository)
- Visual Studio Code (recommended)

---

## Clone the Repository

```bash
git clone https://github.com/Udaykalse/udayfoodie-landing-page.git
```

---

## Navigate to the Project Directory

```bash
cd udayfoodie-landing-page
```

---

## Project Structure

After cloning, the project structure should resemble:

```text
udayfoodie-landing-page/
├── assets/
├── docs/
├── images/
├── index.html
├── style.css
└── README.md
```

---

## Run the Project

Since this is a static HTML project, no installation or build process is required.

Open the project using one of the following methods:

### Method 1 (Recommended)

Open `index.html` in your preferred web browser.

### Method 2 (VS Code Live Server)

1. Install the **Live Server** extension.
2. Open the project in Visual Studio Code.
3. Right-click `index.html`.
4. Select **Open with Live Server**.

---

## Verify Installation

The installation is successful if:

- The homepage loads correctly.
- Images are displayed.
- CSS styling is applied.
- The layout is responsive.

---

## Troubleshooting

If the page appears without styles:

- Verify that `style.css` exists.
- Ensure the stylesheet path in `index.html` is correct.
- Refresh the browser.

If images do not appear:

- Confirm that the `assets/Images/` directory exists.
- Check that image filenames match those referenced in the HTML.
