# Folder Structure

## Overview

This document explains the directory layout of the **UdayFoodie Landing Page** project and the purpose of each file and folder.

---

## Directory Tree

```text
udayfoodie-landing-page/
├── assets/
│   └── Images/
│       ├── delivery__img.webp
│       ├── Fresh-ingredients2.webp
│       ├── ian-dooley-d1UPkiFd04A-unsplash.jpg
│       ├── img-1.webp

├── images/
├── index.html
├── style.css
└── README.md
```

---

## Root Directory

| File/Folder | Description |
|-------------|-------------|
| `README.md` | Main project documentation and entry point for GitHub visitors. |
| `index.html` | Main HTML page that defines the website structure. |
| `style.css` | Contains all styling rules for the website. |
| `assets/` | Stores static assets used by the website. |
| `docs/` | Contains detailed project documentation. |
| `images/` | Stores screenshots used in documentation. |

---

## assets/

The `assets` directory stores static resources used by the website.

### Images/

This folder contains all images displayed on the landing page.

Examples include:

- Hero images
- Food images
- Delivery illustrations
- Background images

Keeping images in a dedicated directory improves project organization and maintainability.

---

## docs/

The `docs` directory contains all technical documentation for the project.

Current documents include:

- Project Overview
- Installation Guide
- Getting Started
- Folder Structure
- Features
- Design
- Assets
- Troubleshooting
- FAQ
- Glossary
- Contributing
- Changelog

---

## images/

This folder stores screenshots used in the documentation.

Current screenshots include:

- Home page
- About section
- Menu section
- Feedback section

These images help readers understand the project visually.

---

## index.html

This is the main entry point of the website.

Responsibilities include:

- Defining page structure
- Organizing sections
- Referencing stylesheets
- Loading image assets

---

## style.css

The stylesheet controls the visual appearance of the project.

Responsibilities include:

- Layout
- Typography
- Colors
- Buttons
- Responsive design
- Spacing

---

## Summary

The project follows a clean and simple directory structure suitable for beginner front-end projects while maintaining separation between source files, assets, documentation, and screenshots.
